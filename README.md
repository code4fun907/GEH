### GEH

this is a simple little script that allows you to run a gif file
as your desktop background

NOTE: I have only tested this on Ubuntu 20.04

# Requirements

- feh
- python 3.9
- Pillow python library

# HOW TO RUN

python main.py gif_name

this will create a .geh directory in your home dir
and prompt you to move the gif into .geh/gifs
then it will start running
